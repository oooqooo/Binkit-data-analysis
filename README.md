This project presents an end-to-end data analysis of Blinkit (formerly Grofers) grocery delivery dataset. The objective is to derive actionable insights into sales performance, customer preferences, and operational efficiency.

The workflow includes data preprocessing, exploratory data analysis (EDA), and visualization using Python (Pandas, NumPy, Matplotlib, Seaborn).

Key insights:

* Identification of top-performing product categories
* Analysis of sales variation across outlet types and locations
* Customer buying patterns and demand trends
* Impact of outlet size and visibility on revenue

This project highlights strong analytical thinking, problem-solving ability, and proficiency in data analysis tools, making it relevant for data analyst roles.
